# Bitcoin-Price-Prediction
Developed a predictive model to forecast Bitcoin prices. Conducted exploratory data analysis (EDA) and experimented with Random Forest and Long Short-Term Memory (LSTM) models. Both models demonstrated good R-squared scores, but LSTM failed to meet the assumptions of hypothesis testing. Consequently, I selected the Random Forest model for deployment. The project was deployed using the Flask framework for real-time predictions.
